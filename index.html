Southern Connecticut Buley Library Bibliography Webpage

The main goal and motivation of this project is to provide the faculty and students the ability to use a user-friendly web interface for accessing important publications. The need for a tool that would allow for an overlay of a better search interface for the Southern Connecticut State Universities publicationâ€™s was self evident. The collection consists of over three thousand publications with some going back to the late fifties which makes navigating through the vast amount of data not easy. With the creation of an interface that provides the user with a user-friendly system, the user is then able to easily navigate through the publications to find the particular publication that the individual desires.
